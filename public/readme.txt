# CIMSS 2026 - Workshop Website

This is the official website for the 6th International Workshop on Critical Infrastructure and Maritime Systems Security.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Open http://localhost:3000

## Deployment

The site automatically deploys to GitHub Pages when you push to the main branch.

See README.md for detailed instructions.
